\doxysubsection{Подробное описание}
Класс для шифрования табличной маршрутной перестановкой 

Реализует шифрование и расшифрование текста методом табличной перестановки с маршрутом записи\+: по горизонтали слева направо, сверху вниз с маршрутом считывания\+: сверху вниз, справа налево
